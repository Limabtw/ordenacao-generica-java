## Ordenação Genérica em Java
## Sobre o projeto

Este projeto foi desenvolvido como atividade prática do curso de Java da EBAC (Escola Britânica de Artes Criativas e Tecnologia).

O objetivo é demonstrar o uso de Generics, da interface Comparable e da classe Collections para realizar a ordenação de objetos em Java de forma reutilizável.

## Tecnologias utilizadas
Java
Generics
Interface Comparable
Classe Collections
## Funcionalidades
Criação da classe Produto com os atributos nome e preco.
Implementação da interface Comparable<Produto> para ordenar produtos pelo preço.
Desenvolvimento de um método genérico para ordenar qualquer lista de objetos comparáveis.
Exibição dos elementos ordenados no console utilizando Collections.sort().

## Objetivo

Praticar os conceitos de ordenação de objetos em Java utilizando a interface Comparable, o uso de métodos genéricos e a classe Collections, desenvolvendo soluções reutilizáveis e organizadas, conforme proposto na atividade da EBAC.
